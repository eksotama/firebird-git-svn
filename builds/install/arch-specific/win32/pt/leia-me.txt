Introdução
==========

Bemvindo à terceira versão Alfa do Firebird 2.0.


Utilizadores Alvo
=================

Esta versão alfa não se destina a uso em ambientes de 
trabalho.
Destina-se apenas a avaliação e tem como objectivo ser 
testada por utilizadores de Firebird experimentados. 
Espera-se que todos os utilizadores leiam primeiro as 
Notas da Versão.

 o NÃO INSTALE esta versão se nunca usou o Firebird primeiro. 
   Se for esse o caso use o Firebird v1.5.2.

 o Se tem vindo a testar versões Alfa anteriores por favor
   consulte o ficheiro de instalação e as Notas da Versão.

Questões conhecidas
===================

 o O processo de instalação falha na detecção dum servidor
   Firebird 2.0 em execução. A Instalação acabará por fracassar
   ao tentar sobrepor um ficheiro que se encontre activo.

 o A base de dados de segurança foi modificada! Consulte 
   a pasta DOC para mais detalhes. Utilizadores que pretendam
   migrar uma base de dados do formato antigo para o novo, poderão
   utilizar o script em misc/upgrade/security

Como Relatar Erros
==================

o Tem a certeza que compreende como o Firebird funciona?

  Talvez se encontre perante um comportamento correcto e 
  a sua questão seja seja apenas uma questão de suporte. 
  Neste caso poderá contactar o suporte-firebird na lista 
  do servidor.

  Poderá subscrevê-la aqui:

    mailto:firebird-support-subscribe@yahoogroups.com

o Ainda assim pensa que se trata dum erro?

  Verifique primeiro a lista de questões sobre testes aqui:

    http://sourceforge.net/tracker/?atid=593943&group_id=9028&func=browse

  É especialmente destinada a serem relatados problemas
  encontrados durante o teste de pre-versões do Firebird.

  Poderá também verificar a lista de Erros Abertos de versões
  estáveis do Firebird. Tal poderá ser encontrada em:

    http://sourceforge.net/tracker/?atid=109028&group_id=9028&func=browse
